Machine Learning

Statistical Models, Neural Nets, and Learning Pipelines

Machine learning projects here include supervised and unsupervised learning systems, vectorization pipelines, statistical inference tools, and neural network applications across domains like forensics, finance, pattern recognition, and linguistic analysis.



  • BitRNG Entropy Predictors
    Predictive ML models for determining entropy patterns from blockchain network data such as mempool size, block timings, and fee rates. Used to inform dice roll simulations and key generation randomness.
  • Duplicate Media Classifier
    An ML-based system to analyze still frames, spectrograms, and metadata from large libraries of video/audio files to find duplicates or degraded variants using CNN, cosine distance, and hashing techniques.
  • MetaTagDB Classifier Engine
    A tagging engine trained on genre, format, and metadata distributions to auto-classify videos, series, and media into structured libraries using probabilistic inference and text analysis of filenames.
  • GKS Behavior Modeling
    Trait evolution and response modeling for Glyph Key Sprites (GKS) using reinforcement learning, training stats based on Lightning Network transaction micro-feeds, and entropy-based progression.
