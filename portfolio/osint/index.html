Open Source Intelligence (OSINT)

Tracking, Mapping, Collection, and Analysis

This section contains tooling and research built for public intelligence collection, field investigations, data mapping, pattern correlation, and adversarial surveillance analytics.



  • tripp.py
    Region-based OSINT tool for routing and dispensary mapping, integrating lodging, food, equipment, and legality layers into multi-day optimized itineraries.
  • OSINT-GeoTrack Engine
    Internal-use platform for mapping digital footprint metadata onto live geographic maps using leaflet.js overlays and EXIF+IP+account co-occurrence.
  • pastebin-indexer.py
    Scraper for pastes across open networks searching for leaked credentials, Bitcoin wallet dumps, compromised endpoints, or keynames.
  • openstreetmap-recon.py
    OSINT mapping GUI using OSM satellite/vector overlays for human terrain, grid intelligence, and POI classification in high-risk zones.
  • discord-surveillance-tools
    Tools for scraping, indexing, and alerting from Discord public servers including keyword monitors, user behavior diffs, and image/video timestamping.
  • Signal/GPG Messaging Audit Scripts
    Intelligence scripts to correlate public GPG keys, contact info, and social metadata across leaked dumps and public footprint scans.
