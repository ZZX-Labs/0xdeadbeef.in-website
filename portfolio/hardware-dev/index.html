Hardware Development

Devices, Microcontrollers, Custom Builds

This section highlights physical hardware projects including custom-built Bitcoin devices, embedded systems, and microcontroller-based hardware platforms using ESP32, RP2040, and STM32 for research, display, automation, and encryption.



  • BitStats Clock
    A real-time Bitcoin block clock using ESP32 and OLED display. Pulls block height, time, and mempool stats with offline caching. Designed for public visual display of the Bitcoin heartbeat.
  • Cryptainer Lock Box
    A hardware device integrating secure cryptainer software with USB-key-based encryption, animated screen feedback, and physical key-splitting logic using multiple microcontrollers.
  • VLC_BOX
    Custom media hardware using Raspberry Pi 5, ProjectM visualizers, and full transport control over OLED and rotary knobs. Designed for standalone audio/video playback in studio settings.
  • ESP32 Mempool Ticker
    A Bitcoin mempool display device using TFT or OLED and Wi-Fi pull from mempool.space API or local node. Displays fee estimates and visualizes congestion state.
  • USB Entropy Key Generator
    Entropy harvesting via soundcard input, USB audio adapter, and analog signals converted to base58 entropy used in secure wallet key generation workflows.
  • GPIO Signal Trainer
    A testbed for low-level bitstream programming and signal logic using multiple GPIOs on RP2040 and a matrix of LED, DAC, speaker, and oscillator responses.
