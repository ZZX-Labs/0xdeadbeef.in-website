Bitcoin Development

Nodes, Tools, Analytics, Hardware, and Protocols

This section explores Bitcoin projects developed for protocol-level interaction, infrastructure deployment, Lightning integration, and hardware wallet design. Emphasis is placed on sovereignty, full-stack control, and privacy-respecting systems.



  • BitTick
    Public real-time Bitcoin ticker engine with global volume-weighted pricing. Integrates over 100 exchange APIs. Includes web interface, REST API, and ESP32-compatible firmware.
  • BitWatch / BitStats
    Hardware-integrated Bitcoin analytics toolset. Displays real-time block height, mempool size, fees, Lightning capacity, and hash rate on ESP32-powered clocks and wristwatches.
  • Bitcoin Full Node Deployment Scripts
    Shell + Python tools to automate `bitcoind`, Tor routing, wallet config, and pruning settings for sovereign Bitcoin stack deployment on ARM/AMD64 systems.
  • Lightning Network Integration
    Includes LND setup automation, invoice handling, LNURL-P support, watchtower setup, and Lightning-native microservices for app-layer integration.
  • Hardware Wallet Design
    Custom firmware and design for physical wallets using QR encoding, inked security seals, and multi-layer physical entropy to produce paper, metal, or plastic wallets.
  • Greybit Tools
    Private-use Bitcoin forensics and intelligence tooling with transaction tracing, address clustering, entity risk analysis, and heatmap-style visualization.
