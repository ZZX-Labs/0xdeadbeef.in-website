Projects, Tools, Frameworks, and Engineering

This section showcases core software projects built by [0xdeadbeef] across cybersecurity, OSINT, Bitcoin automation, AI tooling, and full-stack system design. Projects are engineered for performance, modularity, and long-term maintainability with an emphasis on sovereign infrastructure.



  • youtubedler_cli.py / youtubedler_gui.py
    Advanced CLI/GUI media downloaders supporting proxychains, VPN rotation, archive control, and user-agent spoofing with full automation.
  • cryptainer.py
    Secure container generator with layered GPG, AES, RSA, and ECDSA encryption. Features animated keyguard creatures (Glyph Key Sprites) and polyhedral entropy logic.
  • audiosieve.py
    A VLC-based music library and tagging system with FFmpeg backend, full metadata editing, PyQt5 interface, and smart playlist export and device routing.
  • bitrng.py + rng-dX suite
    Bitcoin RNG toolkit using block height, mempool state, weather oracles, and fee data to seed deterministic or dice-based entropy models.
  • breachcheck.py
    GUI for breach password detection across multiple wordlists. Includes progress bars, index detection, rainbow table attack preparation, and unique list tagging.
  • tripp.py
    OSINT-powered route planning system with support for dispensary search, travel optimization, gear prep, lodging, and cannabis compliance tracking across states.
