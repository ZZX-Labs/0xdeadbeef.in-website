Web Development

Infrastructure, Interfaces, Security, and Design

This section features backend-first and security-focused web development projects. Work spans frontend interfaces, Flask-based APIs, NGINX deployment, Bitcoin-integrated payment systems, admin dashboards, and mirrored `.onion` sites for anonymous access.



  • zzx-labs.io
    Main R&D lab portal with full HTML/CSS/JS frontend, Flask backend, Bitcoin-only payments, Tor mirror, and static+dynamic routing via NGINX.
  • bit-tick.in
    Real-time Bitcoin ticker platform with live price feeds from >100 exchanges, JSON API, calculator, and ESP32 hardware sync compatibility.
  • 0xdeadbeef.in
    Portfolio hub with dynamic templating, file uploads, secure admin panels, and modular asset routing via Python, Jinja, and proxychains for data validation.
  • Private Admin Panels
    Custom-built dashboards for internal control of research uploads, encrypted email portal routing, file management, and remote SFTP control. Optional PGP key auth.
  • Donations & Payments
    Bitcoin-only transactional backend using Flask and QR signing. Generates disposable xpub-linked addresses. Fully open-source and portable for reuse.
  • Mirrored Domains
    Support for `.onion` and `.io` mirror deployment with Cloudflare, certbot SSL, and NGINX auto-sync. Automated failover for surface ↔ dark routing.
