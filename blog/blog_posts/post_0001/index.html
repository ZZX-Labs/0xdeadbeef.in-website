Publication Format

Plaintext-first and independently readable.

The canonical article body is stored in post.txt. This page loads that file and renders supported plaintext formatting into the shared website layout.

The article remains directly accessible even if the rendering script fails, because the source text can always be opened independently.

Metadata is stored separately in metadata.json, allowing the title, publication date, category, tags, revision, license, hashes, signatures, and source links to be updated without editing the page structure.

This same index.html may be reused for future post directories without changing its internal code.
